# Rust Learning
 
 Rust tilini bosqichma-bosqich o‘rganish uchun tartiblangan repo.
 Materiallar **kunma-kun (day-based)** darslar ko‘rinishida `lessons/` ichida yuritiladi.
 
 ## Quick start
 
 ```bash
 cargo run
 ```
 
 ## Requirements
 
 - Rust toolchain: `rustup`, `rustc`, `cargo`
 - Tekshirish:
 
 ```bash
 rustc --version
 cargo --version
 ```
 
 ## Repository structure
 
 - `src/` — hozirgi asosiy demo loyiha (Rust binary)
 - `Cargo.toml` — project manifest
 - `lessons/` — darslar (har bir lesson alohida papka + `NOTES.md`)
 
## Lessons

- Lesson 01: `lessons/lesson-01/NOTES.md`

Umumiy indeks: `lessons/README.md`
 
 ## How to study
 
 - Har bir lesson uchun `lessons/lesson-XX/NOTES.md`ni o‘qing
 - Vazifalarni bajaring
 - Kerak bo‘lsa code’ni kichik commit’lar bilan kiriting
